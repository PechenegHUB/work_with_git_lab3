# Лабораторная работа: Система контроля версий Git

## Описание
В репозитории лежат файлы, ставшие результатом работы с Git, работы с возникшеми конфликтами и тд. В ходе увлекательной работы с ветками, коммитами были получен файл `cursor.txt`. Была сохранена история всех коммитов в файл `history.txt`

## Структура репозитория
- `cursor.txt` — основной файл, используемый для внесения изменений в рамках практической работы.  
- `diff.txt` — файл с сохранённым выводом разницы между коммитами.  
- `history.txt` — файл с историей коммитов.  
- `README.md` — данный файл с описанием лабораторной работы.
